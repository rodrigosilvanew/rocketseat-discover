# Tipos de dados

Conforme o ECMAScript standart temos 9 tipos de dados:

* Data Types:
    * Primitives / Primitive Value
    * Structural
    * Structural Primitive

## Primitivos (não é um objeto e seu valor é imutável)

* String
* Number
* Boolean
* Undefined
* Symbol
* BigInt

# Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function

## Primitivo Estrutural / Structural Root Primitive

* null
